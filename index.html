SciViews provides a series of open source software to supplement R, for statistical computing in a reproductive workflow. The SciViews Box is a completely configured virtual machine that powers scientific processing in a reproducible software environment. Several R packages extend and enhance R. Plugins and custom settings for R Studio, Komodo Edit/IDE, LyX and Github ease and streamline your work. See the about page to read more about SciViews...
